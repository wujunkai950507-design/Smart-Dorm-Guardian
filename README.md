# 小套房智慧守門員平台（Smart Dorm Guardian Platform）

## 一、作品簡介

本作品以「小套房 / 宿舍安全」為主題，設計一個智慧監控儀表板，用來觀察環境狀態並評估危險程度。系統模擬接收多種感測器數據（溫度、瓦斯濃度、煙霧偵測、人體感測等），並透過自訂演算法計算「危險分數」，以視覺化方式提醒使用者目前環境是否安全。

此平台可視為未來實際 IoT 系統的前端介面，搭配 ESP32 等感測裝置，即可應用在學生宿舍、小套房、實驗室等空間。

---

## 二、系統功能

1. **即時狀態面板**
   - 顯示目前溫度、瓦斯指數、煙霧狀態、是否有人活動等模擬數值
   - 危險分數 0–100 分，並以顏色區分安全等級

2. **危險分數演算法**
   - 依照溫度、瓦斯、煙霧與人體感測結果，計算綜合危險分數
   - 危險分數區間：
     - 0–40：安全（綠色）
     - 40–70：注意（黃色）
     - 70–100：危險（紅色）

3. **歷史走勢圖**
   - 利用 Chart.js 顯示最近數筆危險分數變化
   - 可觀察環境風險是否持續上升

4. **事件紀錄（Log）**
   - 當危險分數超過門檻時，記錄一則「警報事件」
   - 顯示時間、分數、簡要說明

5. **模擬資料更新**
   - 透過按鈕或自動更新，隨機產生新一筆感測資料
   - 模擬實際感測器每隔一段時間上傳數值的情境

---

## 三、技術重點

- 前端技術：HTML5、CSS3、原生 JavaScript
- 圖表繪製：Chart.js（以 CDN 方式匯入）
- 資料結構：使用 JavaScript 物件與陣列儲存感測數據與事件紀錄
- 邏輯設計：自訂危險分數演算法與警報規則
- 部署方式：可透過 GitHub Pages 直接發佈為網頁作品
